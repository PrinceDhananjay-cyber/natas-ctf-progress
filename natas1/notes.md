# Natas 1 - Notes

## Concepts Learned

* HTML Source Code
* Browser Developer Tools
* Client-side vs Server-side

## What I Learned

* Disabling right-click does not secure a webpage.
* Hidden information can still be viewed through the page source.
* Sensitive data should never be stored in client-side HTML.

## Key Takeaway

Always inspect the page source when analyzing a webpage. If data is sent to the browser, users can usually access it.
